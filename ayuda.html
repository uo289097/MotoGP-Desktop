Ayuda de MotoGP-Desktop

Inicio

La página Inicio contiene dos bloques de contenido cargados mediante JavaScript. El primer bloque muestra un carrusel de imágenes del circuito de Mandalika obtenido desde la API pública de Flickr, rotando automáticamente cada 3 s. El segundo bloque muestra noticias sobre MotoGP consultando la API de TheNewsAPI, generando artículos con título, entradilla, enlace y fuente.

Piloto

Muestra la información completa sobre un piloto de MotoGP, en este caso Joan Mir. Incluye una descripción general, datos personales, una imagen y un vídeo. También presenta sus equipos a lo largo de su carrera, un glosario de términos de MotoGP y sus resultados en la temporada 2024. Además, ofrece enlaces a sus redes sociales y un apartado con información adicional sobre su trayectoria deportiva.

Circuito

La página permite cargar tres tipos de archivos relacionados con un circuito de MotoGP: un HTML con información del circuito, un SVG con la altimetría y un KML para generar el mapa dinámico. Se comprueba si el navegador soporta API File y, si es así, lee el HTML, lo analiza y lo inserta en la página. Además, se muestra el SVG de altimetría dentro de la sección correspondiente y se procesa un archivo KML, se extraen coordenadas y se genera un mapa interactivo con Mapbox mostrando el trazado del circuito. Todo el contenido se carga dinámicamente según los archivos seleccionados por el usuario.

Para cargar los archivos, pulsa sobre el botón seleccionar archivo. Una vez dentro, sitúate en la carpeta MotoGP-Desktop/xml y selecciona el archivo. En el caso de la información del circuito, el archivo es InfoCircuito.html, en el caso de la altimetría, el archivo se llama altimetria.svg y en el caso del mapa dinámico, el archivo a seleccionar se llama circuito.kml.

Metereología

La página muestra la meteorología de la ciudad más cercana al circuito de MotoGP de Mandalika. Se muestra tanto la información de la ciudad con su getilicio, población y coordenadas como los datos meteorológicos de la carrera, incluyendo temperatura, sensación térmica, humedad, lluvia, velocidad y dirección del viento, hora del amanecer y hora del atardecer. Además, también se pueden ver la media de los datos meteorológicos de los entrenamientos, entre los que se incluyen la temperatura, la humedad, la lluvia y el viento. Toda la información se carga dinámicamente en la página usando la API meteorológica y se presenta en secciones separadas para la carrera y los entrenamientos.

Clasificaciones

En esta página se muestran las clasificaciones de la temporada 2025 de MotoGP tras la carrera de Mandalika. Se indica el piloto vencedor y la duración de la carrera en minutos y segundos. También se muestra la clasificación general con los puntos de los tres primeros clasificados tras la carrera de Mandalika. La información se obtiene de un archivo XML.

Juegos

Presenta los minijuegos disponibles en el proyecto MotoGP Desktop. Incluye un menú con acceso a tres actividades: un juego de memoria con cartas, un cronómetro con JS y un cronómetro desarrollado en PHP. Sirve como página de selección para que el usuario elija y acceda rápidamente a cada juego.

Memoria

Muestra el juego de cartas del proyecto MotoGP Desktop. Presenta un cronómetro y doce cartas emparejadas con logos de distintas marcas, así como el logo de MotoGP que se mezclan aleatoriamente al iniciar la partida. El usuario debe encontrar las parejas volteando las cartas, mientras el temporizador avanza. El juego finaliza cuando todas las cartas quedan reveladas y el cronómetro se detiene.

Cronómetro

Ofrece una herramienta para medir tiempo dentro del proyecto MotoGP Desktop. Muestra un temporizador en pantalla y tres botones: Arrancar, Parar y Reiniciar. El cronómetro actualiza el tiempo cada décima de segundo y lo muestra. El usuario puede controlar en todo momento el inicio, detención y reinicio del conteo.

Cronómetro PHP

Muestra una versión del cronómetro implementada mediante PHP. Permite arrancar, parar y mostrar el tiempo transcurrido usando un formulario con tres botones. El cronómetro calcula el tiempo y la página muestra el resultado en formato mm:ss.s. Para jugar, pulsa el botón arrancar, transcurrido un tiempo pulsa el botón parar y, finalmente, pulsa el botón mostrar para ver el tiempo que ha pasado.
